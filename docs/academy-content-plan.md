# 燃渡学院首页内容方案

> 用途：重构 `src/app/academy/page.tsx` 的内容蓝本
> 平台定位：燃渡AI 是一个 AI 工作流服务平台，提供 Coze 工作流执行、Seedance 视频生成、Seedream 文生图、豆包文案生成等 AI 能力。
> 设计原则：所有内容紧贴平台真实工作流，避免泛泛而谈的"AI 通识"。

---

## 研究结论：主流 AI 教育平台的共性板块

综合极客时间、慕课网、网易云课堂、DataCamp、Coursera、waytoAGI / 通往 AGI 之路的首页结构，提炼出以下高频板块：

| 板块 | 极客时间 | 慕课网 | DataCamp / Coursera | waytoAGI |
| --- | --- | --- | --- | --- |
| Hero（标题+副标题+CTA） | ✅ | ✅ | ✅ | ✅ |
| 学习路径 / 职业路径 | ✅ 体系课 | ✅ 职业路径 | ✅ Career Track | ✅ 布鲁姆分类 |
| 推荐课程 / 精选 | ✅ 专栏+视频课 | ✅ 实战课 | ✅ 推荐 Course | — |
| 讲师介绍 | ✅ 名师大咖 | ✅ 一线讲师 | ✅ Instructor | — |
| 学习数据统计 | ✅ 学员数 | ✅ 2500万用户 | ✅ Learner Count | — |
| 学员评价 | ✅ 口碑 | ✅ 评价 | ✅ Review | — |
| FAQ | ✅ | ✅ | ✅ | ✅ 常见问题 |
| 实战项目 / 动手实践 | ✅ 训练营 | ✅ 实战项目 | ✅ Project | ✅ 案例 |

**结论**：燃渡学院首页应在保留"立即可用教程入口"的基础上，新增 Hero、学习路径、推荐课程、讲师、学习数据、学员评价、FAQ 七大板块，覆盖从"吸引 → 导学 → 信任 → 转化"的完整链路。

## 二、学习路径板块

### 板块标题
- **主标题**：按方向选择你的学习路径
- **副标题**：6 条路径覆盖燃渡AI 平台全部能力，从入门到能独立搭建工作流，每条路径 4–6 个阶段，配套真实工作流练习。

### 板块说明
每条路径卡片包含：路径名 / 适用人群 / 阶段数 / 预计学时 / 涉及能力标签 / 「查看路径详情」入口。

---

#### 路径 1：AI 视频创作实战路径

- **适用人群**：短视频创作者、自媒体运营、电商带货、影视剪辑师
- **核心能力**：Seedance 视频生成 · 首尾帧设计 · 提示词编写 · 视频拼接
- **阶段数**：5 阶段 ｜ 预计学时：12 小时
- **步骤**：
  1. **入门｜认识 Seedance 与 AI 视频生成**：理解文生视频、图生视频、首尾帧控制的差异，跑通第一个工作流（15 分钟出片）。
  2. **进阶｜提示词与镜头语言**：用运镜、景别、光影描述词控制画面节奏，避免"AI 味"。
  3. **实战｜首尾帧设计与过渡**：用 Seedream 出首帧 + 尾帧，喂给 Seedance 生成连贯镜头。
  4. **组合｜多镜头拼接与配音**：结合豆包生成解说文案，用工作流批量生成 + 拼接成完整短视频。
  5. **项目｜从 0 到 1 产出一条带货短视频**：选题 → 文案 → 分镜 → 出图 → 出片 → 字幕，全流程独立完成。
- **结束能力**：能独立用燃渡平台工作流产出一条可发布的 AI 短视频。

---

#### 路径 2：AI 图像设计路径

- **适用人群**：设计师、电商运营、新媒体编辑、品牌策划
- **核心能力**：Seedream 文生图 · 风格控制 · 批量出图 · 商业海报
- **阶段数**：4 阶段 ｜ 预计学时：8 小时
- **步骤**：
  1. **入门｜Seedream 文生图基础**：理解提示词结构（主体 + 风格 + 构图 + 光影），生成第一张可用图。
  2. **进阶｜风格与一致性控制**：固定角色 / IP 形象一致性，学会用 reference 图控制画风。
  3. **实战｜商业场景出图**：商品主图、电商详情页配图、公众号封面、小红书首图的批量产出。
  4. **项目｜为虚构品牌出一套完整视觉物料**：Logo 占位图、主视觉、Banner、社媒配图统一风格。
- **结束能力**：能用 Seedream 工作流稳定产出商用级图像。

---

#### 路径 3：AI 内容创作路径

- **适用人群**：新媒体编辑、文案、内容运营、电商文案
- **核心能力**：豆包文案生成 · 提示词模板 · 多平台改写
- **阶段数**：4 阶段 ｜ 预计学时：6 小时
- **步骤**：
  1. **入门｜豆包文案能力地图**：理解豆包擅长 / 不擅长的场景，建立"输入约束 → 输出可控"的心智。
  2. **进阶｜提示词模板工程**：用角色设定 + 任务描述 + 示例Few-shot，让文案稳定在指定调性。
  3. **实战｜多平台一稿多投改写**：一篇文章改写成公众号 / 小红书 / 抖音文案 / 朋友圈 4 个版本。
  4. **项目｜搭建一个"选题→大纲→正文→标题"的自动化文案工作流**：输入关键词即出 3 篇可发布文案。
- **结束能力**：能搭建自己的内容生产工作流，把日常文案效率提升 5 倍以上。

---

#### 路径 4：Coze 工作流开发路径

- **适用人群**：产品经理、运营、独立开发者、AI 实践者
- **核心能力**：Coze 节点编排 · 大模型节点 · 插件调用 · 工作流调试
- **阶段数**：6 阶段 ｜ 预计学时：15 小时
- **步骤**：
  1. **入门｜Coze 工作流是什么**：理解开始节点、大模型节点、插件节点、结束节点的协作关系。
  2. **进阶｜大模型节点调参**：模型选择（豆包 / Doubao）、温度、system prompt 的实战调优。
  3. **进阶｜插件与工具调用**：调用搜索、图像生成、数据库等插件，打通"思考 → 行动"闭环。
  4. **进阶｜分支与循环**：用条件节点 + 循环节点处理批量任务与异常分支。
  5. **实战｜调试与版本管理**：单节点调试、断点、回滚，避免"改一处崩全线"。
  6. **项目｜从 0 搭建一个可对外发布的工作流**：需求拆解 → 节点设计 → 调试 → 上线 → 在燃渡平台发布使用。
- **结束能力**：能独立设计、搭建、调试、上线一个 Coze 工作流。

---

#### 路径 5：AI 自动化运营路径

- **适用人群**：私域运营、社群运营、电商店长、SaaS 运营
- **核心能力**：自动化运营工作流 · 多步骤编排 · 定时触发 · 数据回流
- **阶段数**：5 阶段 ｜ 预计学时：10 小时
- **步骤**：
  1. **入门｜识别可自动化的运营场景**：客户答疑、日报生成、标签打分、定时触达的选型判断。
  2. **进阶｜多步骤工作流编排**：抓数据 → 豆包分析 → 生成文案 → 输出到指定渠道。
  3. **实战｜智能客服工作流**：意图识别 + 知识库检索 + 兜底回复，覆盖 80% 高频问题。
  4. **实战｜日报 / 周报自动化**：从数据源到结构化日报，10 分钟出一份带图表的运营周报。
  5. **项目｜搭建一条端到端自动化运营流水线**：选自己工作中的真实场景，把重复劳动交给工作流。
- **结束能力**：能把自己工作中至少 1 项重复任务彻底自动化。

---

#### 路径 6：AI 智能对话应用路径

- **适用人群**：客服负责人、产品经理、知识库管理员
- **核心能力**：智能对话 · RAG 检索增强 · 知识库 · 多轮对话
- **阶段数**：4 阶段 ｜ 预计学时：8 小时
- **步骤**：
  1. **入门｜智能对话 vs 普通对话**：理解 RAG、记忆、工具调用在对话中的作用。
  2. **进阶｜知识库构建**：文档切分、向量化、召回调优，让对话"说人话且不出错"。
  3. **实战｜多轮对话与上下文管理**：槽位填充、意图跳转、主动追问的工程化处理。
  4. **项目｜为一个真实业务场景搭建智能客服 Bot**：导入知识库 → 调优 prompt → 压测 → 上线。
- **结束能力**：能搭出一个"问不倒"的智能对话应用。

### 板块标题

#### Q1：我没有编程基础，能学会吗？
**A**：完全可以。燃渡学院所有路径都从"零基础入门"阶段开始，前两个阶段不写任何代码，全程通过燃渡平台可视化界面 + Coze 拖拽编排完成。Coze 工作流开发路径的后半段会涉及少量参数与逻辑，但不要求编程经验，跟着视频操作即可。

#### Q2：教程里的工作流真的能直接跑吗？要不要额外付费？
**A**：所有图文 / 视频教程配套的工作流均上架在燃渡平台，注册账号后即可在工作台直接调用。基础调用免费，超出免费额度后按平台套餐计费，套餐详情见 `/dashboard/orders`。教程本身不收费。

#### Q3：学完一条路径大概要多久？
**A**：每条路径的预计学时已标注在路径卡片上，范围在 6–15 小时之间。建议按阶段分散学习，每个阶段配套一个小练习，平均 1 周可完成一条路径。视频教程支持倍速与回看，节奏可自行控制。

#### Q4：学完能达到什么水平？能直接用来赚钱吗？
**A**：每条路径末尾都设有"实战项目"阶段，完成项目即意味着你能独立产出一份可交付的作品（一条短视频、一套图像物料、一个可上线的工作流等）。是否能变现取决于你自己的应用场景与运营能力，平台提供能力，不承诺收益。

#### Q5：教程内容会更新吗？模型升级了怎么办？
**A**：会持续更新。燃渡平台工作流与底层模型（Seedance / Seedream / 豆包）同步迭代，每次模型升级后我们会同步更新对应图文与视频教程，老学员可免费查看更新内容。学习数据统计板块每月 1 日更新一次。

#### Q6：支持团队 / 企业一起学吗？
**A**：当前首页面向个人学员。团队学习、私有工作流部署、定制培训等需求请通过工作台「联系我们」入口沟通，我们会提供企业版方案。

#### Q7：我已经会用一些 AI 工具了，能跳过入门直接学进阶吗？
**A**：可以。每条路径的阶段均支持单独学习，你可根据自评跳过入门阶段，直接从进阶或实战阶段开始。但建议至少浏览入门阶段的"能力地图"，确认没有知识盲区。

#### Q8：学完后能拿到证书吗？
**A**：暂不提供结业证书。我们更看重"能否产出可交付作品"而非证书，每条路径的实战项目即为你能力的最好证明，作品可直接展示在你的作品集或社交平台。
