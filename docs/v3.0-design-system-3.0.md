# 燃渡AI平台 v3.0 设计系统规范

> 版本: 3.0  
> 创建日期: 2026-07-01  
> 参考竞品: Claude (米色系)、Kimi (极简留白)、Dify (叙事流)

---

## 一、设计理念

### 1.1 核心原则

1. **温暖可信**：区别于冷冰冰的蓝紫科技感，传达"人人都能学会 AI"的亲和力
2. **极简留白**：参考 Kimi/Claude，减少信息密度，让用户聚焦核心动作
3. **叙事引导**：参考 Dify 的叙事流，让用户自然滑入下一步
4. **大厂品质**：参考豆包/Kimi 的精致感，每一个像素都体现专业

### 1.2 差异化定位

| 竞品 | 配色 | 调性 | 燃渡 v3.0 |
|------|------|------|-----------|
| 豆包 | 蓝 #4A90E2 | 活泼年轻 | — |
| 扣子 | 蓝紫渐变 | 科技专业 | — |
| Claude | 米色 #F5F0E8 | 温暖学术 | ✅ 主色调 |
| Kimi | 黑白 | 极简高端 | ✅ 留白参考 |
| Dify | 橙绿点缀 | 开发者友好 | ✅ 叙事流参考 |
| 通义千问 | 深蓝 | 企业专业 | — |

## 六、动效规范

### 6.1 时间系统

| 级别 | 时长 | CSS 变量 | 用途 |
|------|------|----------|------|
| fast | 150ms | `--transition-fast` | 按钮悬停、状态切换 |
| base | 200ms | `--transition-base` | 卡片悬停、下拉展开 |
| slow | 400ms | `--transition-slow` | 页面过渡、模态框 |
| reveal | 600ms | `--transition-reveal` | 滚动揭示动画 |

### 6.2 缓动曲线

```css
/* 标准缓动 */
--ease-out: cubic-bezier(0.16, 1, 0.3, 1);

/* 弹性缓动（按钮点击反馈）*/
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);

/* 滑入缓动 */
--ease-slide: cubic-bezier(0.4, 0, 0.2, 1);
```

### 6.3 滚动揭示动画

```css
/* 交错淡入（stagger）*/
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 触发条件 */
.animate-on-scroll {
  opacity: 0;
  transition: opacity 0.6s ease-out, transform 0.6s ease-out;
}
.animate-on-scroll.visible {
  opacity: 1;
  transform: translateY(0);
}
```

## 七、组件规范

### 7.1 按钮

| 类型 | 背景 | 文字 | 边框 | 用途 |
|------|------|------|------|------|
| Primary | `--color-accent` | 白 | 无 | 主要 CTA |
| Secondary | 透明 | `--color-accent` | 1px `--color-accent` | 次要 CTA |
| Ghost | 透明 | `--color-text` | 无 | 辅助操作 |
| Danger | `--color-error` | 白 | 无 | 危险操作 |

### 7.2 卡片

- 背景：`--color-surface`（白色）
- 圆角：`--radius-md`（12px）
- 阴影：`--shadow-sm`，悬停 `--shadow-hover`
- 内边距：`--space-lg`（24px）
- 过渡：`transition: all var(--transition-base)`

### 7.3 标签/徽章

```css
.badge {
  display: inline-flex;
  align-items: center;
  padding: 4px 10px;
  border-radius: var(--radius-sm);
  font-size: var(--text-caption);
  font-weight: 500;
}

/* NEW 标签（参考火山引擎）*/
.badge-new {
  background: var(--color-new);
  color: white;
  font-size: 11px;
  padding: 2px 6px;
}
```

### 7.4 输入框

- 背景：`--color-surface`
- 边框：`1px solid --color-border`
- 圆角：`--radius-md`
- 聚焦：边框变 `--color-accent`，加 `0 0 0 3px rgba(230,126,34,0.1)` 外发光
- 过渡：`transition: all var(--transition-fast)`
