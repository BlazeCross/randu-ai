# 燃渡AI平台 v3.0 用户画像深度分析报告

> 版本: 3.0  
> 分析日期: 2026-07-01

---

## 一、用户画像总览

### 核心用户群体

燃渡AI平台服务两大核心用户群体：**AI 学员**和**工作流使用者**。

两者有交叉但需求重心不同：
- **AI 学员**：重心在"学习"，关注教程质量、学习路径、实战效果
- **工作流使用者**：重心在"效率"，关注工具易用性、生成质量、操作速度

---

## 二、用户群体一：AI 学员

### 2.1 基本特征

| 维度 | 描述 |
|------|------|
| **年龄** | 18-45 岁，核心人群 22-35 岁 |
| **性别** | 男女比例约 6:4（根据行业数据，AI 工具男性用户略多） |
| **职业** | 电商运营（25%）、新媒体编辑（20%）、设计师（18%）、产品经理（12%）、学生（15%）、创业者/自由职业（10%） |
| **地域** | 一线/新一线城市为主（60%），二三线城市增长快 |
| **教育背景** | 大专及以上（85%），对新技术有好奇心 |
| **技术背景** | 零基础为主（60%），部分有基础编程知识（40%） |

### 2.2 核心诉求（按优先级排序）

1. **系统化学习 AI 技能**：想从零开始系统学习，不想东学一点西学一点
2. **实战导向**：学完能用，能解决工作中的实际问题（如生成电商主图、写短视频脚本）
3. **降低试错成本**：不想花冤枉钱买"割韭菜"课程
4. **快速看到效果**：希望第一次使用就能生成有价值的内容
5. **跟上趋势**：AI 浪潮下不想被淘汰，有职业焦虑

### 2.3 核心痛点

| 痛点 | 描述 | 在燃渡的解决方案 |
|------|------|-----------------|
| **不知道从哪学起** | AI 知识体系庞大，不知从何入手 | 学习路径，按职业/场景设计 |
| **教程与工具脱节** | 看了很多教程但不会用 | 教程配套实际工具，边学边练 |
| **效果不稳定** | AI 生成内容质量忽高忽低 | 预设 Bot/工作流保证效果下限 |
| **怕被割韭菜** | 担心买了课程没效果 | 免费试用 + 透明定价 |
| **时间碎片化** | 通勤、午休才能学习 | 移动端适配，短教程设计 |
| **缺乏反馈** | 不知道学得对不对 | AI 对话式答疑 + 进度追踪 |

### 2.4 使用场景

| 场景 | 具体描述 | 典型时间 |
|------|----------|----------|
| **通勤学习** | 地铁上用手机看教程、学 AI 知识 | 工作日早上 8-9 点，晚上 6-8 点 |
| **午休实战** | 午休时间用 AI 生成内容、快速出图 | 工作日中午 12-14 点 |
| **周末系统学习** | 周末抽 2-3 小时系统学习一个技能 | 周末上午/下午 |
| **工作中遇到问题** | 工作时遇到具体问题，用 AI 快速解决 | 工作日不定时 |
| **副业探索** | 下班后研究 AI 副业变现 | 工作日晚上 8-10 点 |

### 2.5 付费意愿与决策因素

| 维度 | 描述 |
|------|------|
| **月均付费上限** | ¥50-200（学生 ¥50，职场人 ¥100-200） |
| **愿意付费的功能** | 专属 Bot 定制 > 学习路径 > 教程内容 > 基础工具 |
| **决策关键** | 成功案例 > 口碑推荐 > 免费试用 > 定价 |
| **退款态度** | 极度抗拒，要求 7 天无理由退款保障 |
| **转化路径** | 免费试用 → 看到效果 → 付费升级 |

### 2.6 信任需求

1. **真实案例**：需要有真实的学员成功案例（生成图/视频/内容效果展示）
2. **透明定价**：不要有隐藏费用，定价要清晰
3. **学习路径清晰**：明确告诉用户学完之后能做什么
4. **安全保障**：正规平台，数据安全，有隐私政策

### 2.7 细分画像

#### 画像 A：电商运营小美（典型学员）

- **基本信息**：25 岁，女，成都，电商运营 2 年，月薪 6000
- **日常需求**：每天需要生成电商主图、修改商品详情页文案、生成小红书种草文
- **使用路径**：注册 → 看教程"AI 电商实战 7 天" → 开始用"电商主图 Bot" → 生成第一张图 → 付费会员
- **核心动机**：减少加班时间，提升主图点击率（KPI）
- **关注点**：生成效果稳定性、操作简便、是否有成功案例
- **付费意愿**：¥99/月可以接受，超过 ¥200 会犹豫

#### 画像 B：设计师阿强（进阶学员）

- **基本信息**：30 岁，男，深圳，UI 设计师，月薪 15000
- **日常需求**：用 AI 辅助设计灵感、生成参考图、快速出概念稿
- **使用路径**：注册 → 试用 AI 生图 → 感觉有意思 → 探索工作流 → 付费解锁高级 Bot
- **核心动机**：提升设计效率，探索 AI 设计工具前沿
- **关注点**：生成质量、AI 和设计工具结合度、定制化能力
- **付费意愿**：¥300/月可以接受，关注性价比

#### 画像 C：大学生小张（零基础学员）

- **基本信息**：21 岁，男，武汉，大三学生，计算机专业
- **日常需求**：学习 AI 基础知识、探索 AI 工具、完成课程作业
- **使用路径**：看到广告 → 注册 → 看"AI 入门 3 天"教程 → 试用 ChatGPT 对话 → 开始学习提示词工程
- **核心动机**：为找工作加分，了解 AI 趋势
- **关注点**：免费内容多少、学习路径是否系统、是否有证书
- **付费意愿**：¥30/月以下，倾向免费内容为主

## 三、用户群体二：工作流使用者

### 3.1 基本特征

| 维度 | 描述 |
|------|------|
| **年龄** | 25-45 岁，核心人群 28-40 岁 |
| **性别** | 男女比例约 7:3（工具用户男性更多） |
| **职业** | 私域运营（22%）、电商店长（25%）、内容创作者（20%）、独立开发者（15%）、企业团队（18%） |
| **地域** | 一线/新一线城市为主（65%），中小企业主占比高 |
| **技术背景** | 熟悉互联网工具，30% 有 API 集成能力，实用主义导向 |
| **收入** | 月收入 8000-30000，付费能力较强 |

### 3.2 核心诉求（按优先级排序）

1. **提升效率**：用 AI 替代重复劳动，节省时间
2. **快速产出**：操作简单，生成快，不需要复杂配置
3. **效果稳定**：生成质量稳定可预期，不要忽好忽坏
4. **降低成本**：比请人便宜，比自己做快
5. **批量处理**：能一次生成多张图/多个文案

### 3.3 核心痛点

| 痛点 | 描述 | 在燃渡的解决方案 |
|------|------|----------|
| **工具复杂难用** | AI 工具配置复杂，学习成本高 | 预设工作流，零配置使用 |
| **效果不稳定** | 同样的 prompt 每次生成效果不同 | 预设模板保证下限 |
| **成本高** | API 调用费用不透明，算不明白账 | 包月订阅制，清晰定价 |
| **无法批量** | 需要一张一张生成，太慢 | 批量处理能力 |
| **不知道选哪个工具** | 市面 AI 工具太多，不知道哪个好 | 一站式平台，多能力聚合 |

### 3.4 使用场景

| 场景 | 具体描述 | 典型时间 |
|------|----------|----------|
| **每日内容生产** | 每天生成 10-20 张电商图/50 条文案 | 工作日早上 8-10 点 |
| **批量生成** | 活动前批量生成一批素材 | 活动前 1-2 天集中使用 |
| **临时需求** | 突然需要一个图/一段文案 | 工作日随时 |
| **效率对比** | 尝试新的工作流看能否提效 | 周末或空闲时间研究 |

### 3.5 付费意愿与决策因素

| 维度 | 描述 |
|------|------|
| **月均付费上限** | ¥200-500（效率工具预算） |
| **愿意付费的功能** | 批量处理 > 高级工作流 > 专属模板 > 基础 Bot |
| **决策关键** | ROI（投入产出比）> 操作简便 > 口碑 |
| **转化路径** | 试用工作流 → 看到效率提升 → 付费解锁高级功能 |
| **流失原因** | 效果不如预期、价格超出价值感 |

### 3.6 信任需求

1. **效果可量化**：能告诉我用这个工具能节省多少时间
2. **稳定可靠**：不会关键时刻掉链子
3. **快速上手**：5 分钟内能跑通第一个工作流
4. **数据安全**：我的内容和数据不会被泄露或滥用

### 3.7 细分画像

#### 画像 D：电商店长大林（典型效率用户）

- **基本信息**：32 岁，男，杭州，淘宝电商店长，经营母婴类目
- **日常需求**：每天需要生成主图、SKU 图、详情页文案、活动 Banner
- **使用路径**：看到广告 → 试用"电商主图工作流" → 10 分钟生成 20 张图 → 付费年卡
- **核心动机**：减少美工外包费用，提升上新速度（从 3 天到 1 天）
- **关注点**：生成速度、批量能力、生成质量稳定性、价格
- **付费意愿**：¥200-300/月，预期能减少 50% 的设计外包费用

#### 画像 E：私域运营小李（运营用户）

- **基本信息**：28 岁，女，上海，私域运营，负责 3 个品牌社群
- **日常需求**：每天生成朋友圈文案、群发消息、种草文章
- **使用路径**：朋友推荐 → 注册 → 试用"私域文案 Bot" → 生成质量满意 → 付费
- **核心动机**：减少内容创作时间，从每天 3 小时降到 1 小时
- **关注点**：文案质量、生成速度、多品牌适配、模板数量
- **付费意愿**：¥150/月，看重模板丰富度

#### 画像 F：内容创作者阿杰（创作者用户）

- **基本信息**：26 岁，男，成都，小红书/B站内容创作者
- **日常需求**：生成视频脚本、封面文案、评论互动文案
- **使用路径**：看到小红书帖子 → 注册 → 试用"视频脚本 Bot" → 效果不错 → 付费
- **核心动机**：提升内容产出量，增加粉丝增长
- **关注点**：内容创意度、文风适配、热点结合能力
- **付费意愿**：¥100/月，关注内容质量而非数量

## 四、用户行为习惯分析

### 4.1 访问路径

```
首次访问路径：
首页 → 被 CTA 吸引 → 尝试注册 → 完成注册 → 
跳转首页 → 不知道干什么 → 离开（流失)

理想路径：
首页 → 被 CTA 吸引 → 冷启动气泡尝试对话 → 
注册（微信扫码 30 秒完成）→ Onboarding 选择身份 → 
定制首页 → 立刻产生价值（生成第一张图/第一段文案）→ 付费
```

### 4.2 留存关键节点

| 节点 | 时间点 | 关键行为 | 流失风险 |
|------|--------|----------|----------|
| **首次使用** | 注册后 5 分钟内 | 完成第一个任务（生成图/对话） | 极高 |
| **首次产生价值** | 注册后 1 小时内 | 拿到第一个满意的结果 | 高 |
| **发现核心功能** | 注册后 1-3 天 | 找到想要的工作流/教程 | 中 |
| **首次付费** | 注册后 7 天内 | 试用到期前决策 | 中 |
| **形成习惯** | 注册后 14 天后 | 每周使用 3 次以上 | 低 |

### 4.3 用户分享动机

| 动机 | 场景 | 分享形式 |
|------|------|----------|
| **炫耀成果** | 生成了特别好看的电商图/视频 | 朋友圈/小红书 |
| **实用分享** | 发现一个好用的工作流 | 社群/朋友圈 |
| **求助交流** | 遇到问题请教 | 微信群/问答社区 |
| **口碑推荐** | 觉得好用推荐朋友 | 口碑介绍 |

## 五、用户认知与关注点

### 5.1 优先级排序

| 关注点 | 权重 | 满足方式 |
|--------|------|----------|
| **效果可预期** | ⭐⭐⭐⭐⭐ | 预设模板 + 案例展示 |
| **学习成本低** | ⭐⭐⭐⭐⭐ | Onboarding + 冷启动引导 |
| **性价比高** | ⭐⭐⭐⭐ | 透明定价 + 免费试用 |
| **操作简便** | ⭐⭐⭐⭐ | 零配置工作流 + 简洁 UI |
| **安全合规** | ⭐⭐⭐ | 隐私政策 + 备案号展示 |
| **定制化能力** | ⭐⭐⭐ | 专属 Bot + 自定义模板 |

### 5.2 信任建立路径

```
信任起点 → 平台感知（设计/品牌）→ 功能试用 → 效果验证 → 口碑参考 → 付费决策
```

- **平台感知（10 秒）**：网站设计专业感、品牌调性
- **功能试用（5 分钟）**：冷启动气泡/免费试用
- **效果验证（1 小时）**：生成第一个满意结果
- **口碑参考（始终）**：真实案例、学员评价、社交媒体内容

## 六、用户研究对 v3.0 设计的指导

### 6.1 登录/注册优化

| 发现 | 设计决策 |
|------|----------|
| 微信扫码降低注册门槛 | 新增微信扫码登录 |
| 首次使用是关键留存点 | Onboarding 引导定制首页 |
| 新用户不知道干什么 | 冷启动气泡引导 |

### 6.2 首页/导航优化

| 发现 | 设计决策 |
|------|----------|
| 学员关注学习路径 | Navbar 新增"学习"导航 |
| 效率用户关注工具 | 保留"工作台"入口 |
| 首次 5 分钟是黄金期 | 首页叙事流快速传达价值主张 |

### 6.3 学习功能优化

| 发现 | 设计决策 |
|------|----------|
| 学员需要系统化学习 | 新增学习路径功能 |
| 怕被割韭菜 | 免费试用 + 成功案例展示 |
| 学习效果需要反馈 | 学习进度追踪 |

### 6.4 定价/转化优化

| 发现 | 设计决策 |
|------|----------|
| 学员付费上限 ¥50-200 | 定价分层，免费+基础+高级 |
| 效率用户看 ROI | 定价页展示"节省 XX 小时" |
| 需要 7 天退款保障 | 定价页加退款说明 |

## 七、结论

燃渡 AI 平台的用户核心特征是**"想要用 AI 解决实际问题，但不知道怎么做"**。

v3.0 的核心设计原则：
1. **低门槛**：微信扫码登录 + 冷启动气泡 + 零配置工作流
2. **快价值**：首次使用 5 分钟内产生价值
3. **清晰路径**：学习路径（学员）/ 工作流分类（效率用户）
4. **信任背书**：真实案例 + 数据展示 + 合规保障
