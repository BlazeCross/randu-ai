# 燃渡AI平台 v3.0 竞品研究报告

> 版本: 3.0  
> 研究日期: 2026-07-01  
> 研究范围: 12 个主流 AI 平台（国内 8 个 + 国际 4 个）

---

## 一、研究方法论

本研究采用以下维度对竞品进行全方位分析：
- **产品定位**：目标用户、核心价值主张
- **首页设计**：Hero、叙事流、CTA
- **登录/注册流程**：注册门槛、登录方式
- **核心功能**：智能体、工作流、学院
- **定价策略**：免费策略、付费分层
- **视觉设计**：配色、字体、动效、交互
- **增长策略**：新用户引导、激活、留存

---

## 二、国内竞品分析

### 1. 豆包（doubao.bytedance.com）

**产品定位**：字节跳动旗下 AI 智能助手，面向 C 端用户的日常 AI 对话工具

| 维度 | 分析 |
|------|------|
| **目标用户** | 大众用户、年轻人、AI 小白 |
| **核心入口** | 首页即对话，无多余 marketing 内容 |
| **首页设计** | 极简设计，大面积留白，对话框居中，冷启动气泡引导 |
| **登录方式** | 手机号 / 抖音一键登录 |
| **差异化亮点** | 对话气泡有品牌 IP（豆包 AI 数字人），字节系 App 一键登录 |
| **视觉风格** | 蓝色主色调（豆包蓝 #4A90E2），圆润卡片，留白多 |
| **学习路径** | 无学院，定位纯工具 |
| **定价策略** | 完全免费，字节补贴策略 |
| **给我们的启发** | 冷启动气泡 + 极简首页 + 社交账号一键登录 = 最低注册门槛 |

### 2. 扣子（coze.cn）

**产品定位**：字节跳动旗下 AI 应用开发平台，面向开发者和企业

| 维度 | 分析 |
|------|------|
| **目标用户** | 开发者、AI 爱好者、企业 |
| **首页设计** | Marketing + Demo 并行，顶部导航 + Hero + Bot 市场 |
| **核心功能** | Bot 市场（卡片墙）、工作流编排、知识库管理 |
| **登录方式** | 手机号 / 抖音账号 |
| **Bot 市场** | 卡片墙布局，每张卡片有头像、名称、使用次数、评分 |
| **视觉风格** | 蓝紫渐变，卡片化布局，标签系统（NEW/HOT） |
| **增长策略** | 平台补贴优秀 Bot 创作者，形成内容护城河 |
| **给我们的启发** | Bot 卡片墙 + 评分/使用次数 = 工作流市场可参考的模板 |

### 3. 通义千问（tongyi.aliyun.com）

**产品定位**：阿里云旗下企业级 AI 大模型，面向企业用户

| 维度 | 分析 |
|------|------|
| **目标用户** | 企业用户、开发者、AI 工程师 |
| **首页设计** | Dashboard 风格，左侧导航 + 右侧对话区，顶部功能区 |
| **核心功能** | 对话、代码助手、创意文案、知识库、企业知识管理 |
| **登录方式** | 阿里云账号 / 钉钉扫码 |
| **视觉风格** | 深蓝色企业风格，简洁专业，企业感强 |
| **定价策略** | 免费额度 + 企业版付费（按 token 计费） |
| **给我们的启发** | 左侧导航 + Dashboard 风格 = 企业用户友好，可用于工作台参考 |

### 4. Kimi（kimi.moonshot.cn）

**产品定位**：月之暗面旗下长文本 AI，面向知识工作者

| 维度 | 分析 |
|------|------|
| **目标用户** | 知识工作者、研究人员、内容创作者 |
| **首页设计** | 极简黑白设计，首页即对话，几乎无 marketing |
| **登录方式** | 手机号 |
| **核心差异化** | 超长上下文（200 万字），PDF/URL/文档上传分析 |
| **视觉风格** | 极简黑白，留白为主，衬线字体（中文优雅），无多余装饰 |
| **给我们的启发** | 极简设计 + 核心能力突出 = 高端感 + 专业感，燃渡可借鉴学院页的简洁风格 |

### 5. 智谱清言（chatglm.cn）

**产品定位**：智谱华章 AI GLM 大模型，面向学术和开发者

| 维度 | 分析 |
|------|------|

| **视觉风格** | 深蓝色，功能标签页式布局，信息密度高 |
| **定价策略** | 免费额度 + 付费会员（月卡/年卡） |

### 6. 讯飞星火（xinghuo.xfyun.cn）

| 维度 | 分析 |

### 7. Dify（dify.ai）

**产品定位**：开源 AI 应用开发平台，面向开发者

## 四、竞品设计共同规律总结

### 视觉设计规律

| 规律 | 具体表现 | 燃渡应用 |
|------|----------|----------|
| 配色差异化 | Claude 米色/Kimi 黑白 vs 国内蓝紫同质化 | v3.0 改为温暖米色系，避免蓝紫 |
| 留白极致化 | Kimi/Claude/Perplexity 大量留白 | 全站加强留白，减少信息密度 |
| 数字背书 | 火山引擎大字号展示用户量 | 首页数据展示增强 |
| NEW 标签 | ChatGPT/火山引擎用 NEW 引导探索 | 新功能加 NEW 标签 |

### 交互设计规律

| 规律 | 具体表现 | 燃渡应用 |
|------|----------|----------|
| 冷启动气泡 | 豆包/ChatGPT 首页气泡引导 | 首页 Hero 区气泡设计 |
| 动词化导航 | Dify 的 BUILD/CONNECT/START | 导航采用动词命名 |
| 极简首页 | Kimi/Perplexity 首页即工具 | 首页重构为叙事流 |
| 功能分层 | ChatGPT/Perplexity Pro 突出 | 定价页分层展示 |

### 增长策略规律

| 规律 | 具体表现 | 燃渡应用 |
|------|----------|----------|
| 社交登录 | 豆包/扣子用抖音一键登录 | 新增微信扫码登录 |
| 低门槛注册 | ChatGPT 邮箱即可 | 微信扫码降低注册门槛 |
| 用户引导 | Notion 引导模板 / ChatGPT NEW | Onboarding 引导流程 |
| 社会证明 | Dify 推文墙 / 火山引擎数据 | 首页学员数量/生成量展示 |

## 五、燃渡AI差异化定位建议

基于竞品分析，燃渡 AI 的差异化定位建议：

| 维度 | 当前 | 竞品参考 | v3.0 调整 |
|------|------|----------|-----------|
| **配色** | 蓝紫渐变 | Claude 米色 | 改为温暖米色系（#FAF7F2 为主） |
| **字体** | 无衬线 | Claude Georgia 中文衬线 | 中文使用优雅衬线（系统默认+回退） |
| **首页** | 轮播 | 豆包极简/Kimi 叙事流 | 改为静态 Hero + 叙事流 |
| **导航** | 名词导航 | Dify 动词导航 | 探索/创作/学习/我的 |
| **登录** | 手机号+邮箱 | 抖音/微信一键 | 新增微信扫码登录 |
| **引导** | 无 | Notion 引导/NEW 标签 | 新增 Onboarding + NEW 标签 |
| **学习** | 教程列表 | 讯飞行业路径 | 新增学习路径场景化 |
| **背书** | 无 | 火山引擎大数字 | 首页展示"已服务 X 位学员" |
