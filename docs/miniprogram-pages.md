# 微信小程序页面结构

## 核心页面

| 页面 | 功能 | 对应网站页面 |
|------|------|-------------|
| 首页 | 功能导览+快速入口 | / |
| AI对话 | 智能体对话 | /workspace |
| 工作流 | 热门工作流 | /marketplace |
| 学习 | 学习路径 | /academy/paths |
| 我的 | 个人中心 | /dashboard |

## 页面详细设计

### 首页
- Logo + 平台名称
- 4个核心功能入口（AI对话/图片/视频/工作流）
- 快捷入口（学AI/用工具）
- 底部导航栏

### AI对话页
- 顶部：Bot选择器
- 中部：消息列表
- 底部：输入框 + 发送按钮

### 工作流页
- 顶部：分类筛选
- 内容：工作流卡片网格
- 卡片：图标 + 名称 + 使用次数

### 学习页
- 学习路径列表
- 路径卡片：标题 + 进度 + 教程数

### 我的页
- 用户头像 + 昵称
- 会员状态
- 学习进度
- 设置入口
