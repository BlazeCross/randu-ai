## 三、合规改造清单

### 3.1 《用户协议》页面（高优先级）

**法律依据**：《个人信息保护法》第 17 条，服务提供者应显著方式告知用户相关事项。

**必须包含的内容**：

```markdown
# 燃渡AI用户协议

一、服务说明
- 燃渡AI是什么平台
- 服务内容（AI对话、图片生成、视频生成、工作流等）
- 服务变更和终止条件

二、用户注册与账号
- 注册条件（手机号实名）
- 账号安全责任
- 账号注销权利

三、用户行为规范
- 禁止内容（违法、违规、侵权、色情等）
- 禁止用途（商业转售、非法牟利等）
- 违规处理措施

四、知识产权
- 平台内容的知识产权归属
- 用户生成内容的权利
- AI 生成内容的归属说明

五、免责声明
- AI 生成内容的准确性免责
- 服务中断的免责
- 不可抗力的免责

六、争议解决
- 适用法律
- 争议解决方式（协商→仲裁/诉讼）

七、协议更新
- 更新通知方式
- 生效时间
```

### 3.2 《隐私政策》页面（高优先级）

**法律依据**：《个人信息保护法》第 17 条，收集个人信息应告知并取得同意。

**必须包含的内容**：

```markdown
# 燃渡AI隐私政策

一、信息收集
- 基础信息（手机号、设备信息、IP 地址）
- 使用信息（对话记录、生成内容、使用时间）
- 第三方信息（微信 UnionID，如使用微信登录）

二、信息使用目的
- 提供 AI 服务
- 账户管理
- 安全审计
- 服务优化
- 法律法规要求

三、信息共享范围
- 不向第三方出售用户信息
- 必要时的共享（法律要求、合并收购）

四、用户权利
- 查阅权（查看个人信息）
- 更正权（修改错误信息）
- 删除权（注销账号删除数据）
- 撤回同意权

五、信息存储
- 存储地点（阿里云成都节点）
- 存储期限（账号注销后 30 天内删除）
- 安全措施（加密存储、HTTPS 传输）

六、未成年人保护
- 不满 14 周岁需监护人同意
- 监护人权利（查阅、删除）

七、Cookie 政策
- 必要 Cookie（登录状态）
- 统计 Cookie（使用分析）
- 用户可禁用 Cookie
```

### 3.3 内容安全机制（高优先级）

**法律依据**：《生成式人工智能服务管理暂行办法》第 12 条，服务提供者应具备防止生成违法内容的技术能力。

**实施步骤**：

1. **接入阿里云内容安全（绿网）API**
   - 控制台：https://yundun.console.aliyun.com/
   - 选择"内容安全"产品
   - 开通文本审核（¥0.15/千次）

2. **在关键接口接入审核**

```typescript
// src/lib/content-check.ts
import crypto from "crypto";

interface CheckResult {
  pass: boolean;
  reason?: string;
}

export async function checkTextContent(text: string): Promise<CheckResult> {
  const accessKeyId = process.env.ALIYUN_ACCESS_KEY_ID!;
  const accessKeySecret = process.env.ALIYUN_ACCESS_KEY_SECRET!;
  
  const requestBody = JSON.stringify({
    scenes: ["antispam"],
    tasks: [{
      content: text,
    }],
  });

  // 阿里云内容安全 API 调用签名
  const signature = signRequest(accessKeyId, accessKeySecret, requestBody);
  
  const response = await fetch(
    "https://green.cn-shanghai.aliyuncs.com/green/text/scan",
    {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        "Authorization": signature,
        "X-ACS-Signature-Method": "HMAC-SHA1",
        "X-ACS-Signature-Version": "1.0",
      },
      body: requestBody,
    }
  );

  const result = await response.json();
  
  // 检查是否有违规内容
  const taskResults = result.data?.[0]?.results || [];
  const hasViolation = taskResults.some(
    (r: any) => r.suggestion === "block"
  );

  if (hasViolation) {
    const reason = taskResults.find(
      (r: any) => r.suggestion === "block"
    )?.reason;
    return { pass: false, reason: reason || "内容违规" };
  }

  return { pass: true };
}
```

3. **接入点**：
   - `/api/chat`：对话内容审核
   - `/api/external/generate/image`：图片生成描述审核
   - `/api/external/generate/video`：视频生成描述审核
   - `/api/external/webhook/volcengine`：回调内容审核

4. **AI 生成内容标注**（推荐）：
   - 在 AI 对话回复末尾添加标注："以上内容由 AI 生成，仅供参考"
   - 在图片/视频生成末尾添加可见水印

### 3.5 敏感操作日志

**法律依据**：《网络安全法》第 21 条，应记录网络运行状态和网络安全事件。

**记录范围**：
- 用户登录/登出（IP、时间、设备）
- 密码修改
- 积分充值/消费
- 管理后台操作（管理员）

**日志字段**：
```typescript
interface SensitiveLog {
  id: string;
  userId: string;
  action: "login" | "logout" | "password_change" | "credits_charge" | "admin_action";
  ip: string;
  userAgent: string;
  timestamp: Date;
  details?: Record<string, any>; // 额外信息
}
```

### 3.7 未成年人保护

**法规依据**：《个人信息保护法》第 31 条，不满 14 周岁的未成年人个人信息视为敏感信息。

**实施措施**：
- 注册时增加年龄确认（18 岁以下需监护人手机号验证）
- 在用户协议中增加未成年人专门条款
- 内容安全审核对涉及未成年人的内容加强审核

## 五、运营合规建议

### 5.1 内容安全运营

- 每日查看阿里云绿网后台的违规统计
- 建立违规内容人工复核机制
- 定期抽查 AI 生成内容质量

### 5.2 数据安全运营

- 每季度检查数据备份完整性
- 阿里云 OSS 设置访问日志
- 定期审计数据库访问权限

### 5.3 应急响应

- 建立违规内容发现→处置→报告流程
- 制定《网络安全事件应急预案》
- 按规定向监管部门报告重大安全事件（一般 7 日内）
